## 🛸 About The Project

> **"In a disaster, seconds matter. ResQ-Link bridges the gap between chaos and coordination."**

**ResQ-Link** is a next-gen disaster response platform designed to streamline communication between victims, volunteers, and official rescue agencies. By leveraging real-time geolocation and resource tracking, we turn scattered cries for help into organized rescue missions.

### 🌟 Why ResQ-Link?
* **Zero-Latency Alerts:** Instant SOS propagation to nearest response units.
* **Visual Command Center:** A master dashboard for agencies to visualize crisis zones.
* **Offline First:** Critical features work even when the internet is spotty.

## ⚡ Getting Started

Ready to save the world? Follow these steps to get a local copy up and running.

### Prerequisites

* **Node.js** (v16+)
* **npm** (v8+)

### 🛠️ Installation

1.  **Clone the Repo**
    ```sh
    git clone [https://github.com/I-AM-A-PROGRAMER/ResQ-Link.git](https://github.com/I-AM-A-PROGRAMER/ResQ-Link.git)
    ```
2.  **Install Dependencies**
    ```sh
    npm install
    ```
3.  **Configure Environment**
    Create a `.env` file in the root directory:
    ```js
    API_KEY = "ENTER_YOUR_API_KEY";
    DB_URI = "ENTER_YOUR_MONGO_URI";
    ```
4.  **Launch**
    ```sh
    npm run dev
    ```
